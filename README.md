## Installation

#### Clone the project

```bash
  git clone https://github.com/AliNikandish/WordleGame.git
```

#### Go to the project directory

```bash
  cd WordleGame
```

#### Install WordleGame with npm

```bash
  npm install
```
#### Run:
```bash
  npm run dev
```
